Put website screenshots into any WordPress post/page with a shortcode and a URL as parameter.

== Description ==

Quickly insert a thumbshot of any website with a shortcode and a URL as a parameter into any WordPress post/page. This plugin makes use of the Thumbshots.com service and can be used with either a free or a paid account.

Thanks to Ian Blackford from <a href="http://www.designconscious.com/">Design Conscious</a> for the initial concept and funding of the plugin.

== Installation ==

Follow the steps below to install the WordPress Thumbshots plugin on your site...

1. Go to the 'Plugins' page of your WordPress dashboard.
1. Click the 'Add New' button at the top to install a new plugin.
1. Search for "wordpress thumbshots plugin" and in the results, find it.
1. Click the 'Install' link to the right and install the plugin.

OR

1. Download the ZIP package from WordPress.org.
1. Upload the `thumbshots` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress

To display a thumbnail/screenshot of a website inside any WordPress post/page, put the shortcode `[thumbshot url="http://domain.com"]` and replace the URL parameter with your preferred URL/link. You can use multiple instances of this shortcode in a post/page and on multiple posts/pages.

== Frequently Asked Questions ==

= Can I use multiple instances? =

Yes, you can use multiple instances on a post/page and on multiple posts/pages.
